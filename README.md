# X4-Scripts

Dieses Projekt enthält Beispielskripte und Ressourcen für X4: Foundations.

## Dokumentation

- `CODE_REFERENCE.md` – kuratierte Übersicht wichtiger Funktionen und Befehle.
- `SCRIPT_COMMAND_REFERENCE.md` – automatisch erzeugte Liste aller in den Skripten verwendeten XML-Befehle mit Vorkommen und Beispieldatei.
